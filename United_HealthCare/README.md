# United Healthcare Member Insights Dashboard

## Purpose
A comprehensive healthcare analytics tool designed to provide deep insights into member engagement, risk management, and healthcare utilization patterns.

## Key Dashboard Components

### 1. Member Spend & Risk Overview

**Key Metrics:**
- Total Members: 29
- High-Risk Members: 21%
- Average Forecast Spend: $7,251
- Churned Members: 6

**Risk Segmentation:**
- High-Risk Spend: $40,625
- Medium-Risk Spend: $65,230
- Low-Risk Spend: $50,918

### 2. Engagement Tracker

**Engagement Insights:**
- Total Activities: 37
- Most Used Channel: Call Center
- Average Engagement per Member: 1.28
- Least Engaged Segment: Wellness Visit

## Technical Implementation

### Data Sources
- Member Management System
- Claims Database
- Engagement Tracking Platform

### Key Analytical Techniques
- Risk Stratification
- Engagement Channel Analysis
- Predictive Spend Modeling

### Technology Stack
- Data Visualization: Tableau
- Data Cleaning : Excel
